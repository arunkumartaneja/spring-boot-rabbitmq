# Spring boot AMQP with rabbitmq

### `Rabbitmq installation` 
- https://www.rabbitmq.com/download.html

### `Features cover in App`
- Simple messaging using queue
- Exchange - Direct, Topic, Fanout
- Message retry
- DLQ and DLX
- Parking lot queue
- Exception handling
